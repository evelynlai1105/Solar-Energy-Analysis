# Solar Energy Analysis

太陽能發電預測分析

---

## Project Overview

本專案利用機器學習分析影響太陽能發電量的重要因素，建立發電量預測模型，提供能源管理與發電效率分析參考。

---

## Business Problem

太陽能發電受到天候、溫度、日照等因素影響。

如何提高預測準確度，是能源管理的重要課題。
